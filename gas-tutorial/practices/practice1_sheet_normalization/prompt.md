# Practice 1 — Sheet 정규화 + 이메일 매칭 (Beginner)

아래 프롬프트를 LLM(ChatGPT/Claude)에 복사해서 입력하세요.

> **팁**: 코드가 어떤 작업을 했는지 추적할 수 있도록, LLM에게 **각 단계마다 로그(Logger.log)를 남겨달라고 요청**하는 것을 권장합니다.

---

```text
Google Apps Script로 스프레드시트 데이터를 정리하고 싶어.

현재 상황:
- 내 스프레드시트에 시트가 2개 있어
- Sheet1: 이름, 나이, 지원트랙, 지원동기, 연락처
- Sheet2: 이름, 이메일

문제점:
- 이름에 공백이 들어가 있는 경우가 있음 (예: "이  준")
- 연락처 형식이 제각각임 (010 1234 5678, 010-1234-5678, +82-10-1234-5678 등)
- Sheet2에 이메일이 있는데, 이름으로 매칭해서 Sheet1 데이터와 합치고 싶음

할 일:
1) 이름 공백 정리
2) 연락처를 010-1234-5678 형식으로 통일
3) Sheet2에서 이름으로 이메일 찾아서 매칭
4) 정리된 결과를 새 시트에 저장
5) 이메일 못 찾은 사람은 표시해줘
6) 몇 명 처리했는지 로그로 알려줘

Google Apps Script 문법으로 작성해줘 (var 사용).
```

---

## 에러가 났을 때

```text
방금 준 코드를 실행했는데 에러가 났어.

[에러 메시지]
여기에 실행 로그 에러를 그대로 붙여넣기

[현재 코드]
여기에 전체 코드를 붙여넣기

수정해줘. 뭐가 문제였는지 간단히 설명도 해줘.
```

## 결과가 예상과 다를 때

```text
코드가 에러는 안 나는데 결과가 이상해.

[예상한 결과]
여기에 기대했던 결과 적기

[실제 결과]
여기에 실제 로그 붙여넣기

왜 다른지 확인하고 수정해줘.
```
