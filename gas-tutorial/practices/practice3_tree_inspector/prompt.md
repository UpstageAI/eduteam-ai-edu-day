# Practice 3 — Drive 구조를 시트에 출력 (Advanced Prep)

아래 프롬프트를 LLM(ChatGPT/Claude)에 복사해서 입력하세요.

> **팁**: 코드가 어떤 작업을 했는지 추적할 수 있도록, LLM에게 **각 단계마다 로그(Logger.log)를 남겨달라고 요청**하는 것을 권장합니다.

---

```text
Google Apps Script로 Drive 폴더 구조를 Google Sheet에 정리해서 보고 싶어.

현재 상황:
- "GAS_Tutorial_Workspace"라는 폴더 안에 하위 폴더가 여러 개 있어
- 어떤 구조인지 파악이 안 돼서 시트에 트리 형태로 보고 싶어

할 일:
1) 폴더 이름으로 루트 폴더를 찾아줘
2) 하위 폴더와 파일을 전부 탐색해서 시트에 적어줘
3) 들여쓰기로 깊이를 표현해줘 (폴더 안의 폴더는 더 안쪽으로)
4) 폴더인지 파일인지 구분이 되면 좋겠어
5) 전체 경로도 같이 보여줘
6) 마지막에 폴더 몇 개, 파일 몇 개인지 요약해줘

Google Apps Script 문법으로 작성해줘 (var 사용).
```

---

## 에러가 났을 때

```text
방금 준 코드를 실행했는데 에러가 났어.

[에러 메시지]
여기에 실행 로그 에러를 그대로 붙여넣기

[현재 코드]
여기에 전체 코드를 붙여넣기

수정해줘. 뭐가 문제였는지 간단히 설명도 해줘.
```

## 결과가 예상과 다를 때

```text
코드가 에러는 안 나는데 결과가 이상해.

[예상한 결과]
여기에 기대했던 결과 적기

[실제 결과]
여기에 실제 로그 붙여넣기

왜 다른지 확인하고 수정해줘.
```
