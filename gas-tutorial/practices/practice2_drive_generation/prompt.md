# Practice 2 — 트랙별 폴더 + 지원자 문서 생성 (Intermediate)

아래 프롬프트를 LLM(ChatGPT/Claude)에 복사해서 입력하세요.

> **팁**: 코드가 어떤 작업을 했는지 추적할 수 있도록, LLM에게 **각 단계마다 로그(Logger.log)를 남겨달라고 요청**하는 것을 권장합니다.

---

```text
Google Apps Script로 Drive에 폴더와 문서를 자동으로 만들고 싶어.

현재 상황:
- 아까 정리한 지원자 데이터가 스프레드시트에 있어
- 시트에 이름, 지원트랙, 지원동기 등이 있어

할 일:
1) Drive에 "트랙별_지원자" 폴더를 만들어줘
2) 지원트랙별로 하위 폴더를 만들어줘 (예: SeSAC, SSAFY 등)
3) 지원자마다 "이름.docx" 문서를 만들어서 해당 트랙 폴더에 넣어줘
4) 문서 안에 지원자 이름, 트랙, 지원동기를 적어줘
5) 같은 이름 파일이 이미 있으면 건너뛰고 로그만 남겨줘
6) 트랙별로 몇 개 만들었는지 요약해줘

Google Apps Script 문법으로 작성해줘 (var 사용).
```

---

## 에러가 났을 때

```text
방금 준 코드를 실행했는데 에러가 났어.

[에러 메시지]
여기에 실행 로그 에러를 그대로 붙여넣기

[현재 코드]
여기에 전체 코드를 붙여넣기

수정해줘. 뭐가 문제였는지 간단히 설명도 해줘.
```

## 결과가 예상과 다를 때

```text
코드가 에러는 안 나는데 결과가 이상해.

[예상한 결과]
여기에 기대했던 결과 적기

[실제 결과]
여기에 실제 로그 붙여넣기

왜 다른지 확인하고 수정해줘.
```
