## Google Apps Script란?

### GAS의 특징

- **Google이 만든 클라우드 기반 자동화 도구**
- JavaScript 기반이지만 **직접 쓸 필요 없음** — AI가 대신 작성해줍니다
- 브라우저에서 바로 실행 — 별도 설치 불필요
- Google 계정만 있으면 **무료**로 사용 가능

### GAS로 제어할 수 있는 서비스

| 서비스 | 할 수 있는 일 |
|--------|--------------|
| Google Sheets | 데이터 정리, 계산, 자동 입력 |
| Google Drive | 폴더 생성, 파일 이동, 구조 파악 |
| Google Docs | 문서 자동 생성, 내용 채우기 |
| Gmail | 이메일 자동 발송, 읽기 |
| Google Calendar | 일정 자동 등록, 조회 |
| Google Forms | 응답 데이터 처리 |

### Apps Script 에디터 열기

1. **Google Sheets** 열기
2. 상단 메뉴 → **확장 프로그램** → **Apps Script** 클릭
3. 에디터가 새 탭에서 열림
4. 여기에 코드를 붙여넣고 **▶ 실행** 버튼 클릭

> **처음 실행하면 권한 승인 팝업이 뜹니다.** '허용'을 누르면 됩니다. 이는 GAS가 내 Google 계정의 데이터에 접근하기 위한 정상적인 절차입니다.

## AI에게 잘 물어보는 법

AI(ChatGPT, Claude)는 질문이 구체적일수록 좋은 코드를 만들어줍니다. 아래 공식을 따르면 대부분의 경우 바로 쓸 수 있는 코드를 받을 수 있습니다.

### 프롬프트 작성 공식

| 순서 | 항목 | 예시 |
|------|------|------|
| 1 | **도구 지정** | "Google Apps Script로" |
| 2 | **현재 상황** | "내 시트에 ~가 있어" |
| 3 | **문제점** | "~이 이상해 / ~이 안 맞아" |
| 4 | **할 일** | 번호로 나열 (1, 2, 3...) |
| 5 | **제약 조건** | "GAS 문법으로, var 사용" |

> **핵심**: 무엇을 하고 싶은지 **구체적으로** 써야 합니다.

### 프롬프트 템플릿

아래 빈칸을 채워서 AI에게 보내세요.

```
도구: Google Apps Script로 ________________ 하고 싶어.

현재 상황:
- 현재 데이터/파일이 어떤 상태인지

문제점:
- 무엇이 잘못되었는지 / 불편한 점

할 일:
1) 구체적인 작업 1
2) 구체적인 작업 2
3) ...

Google Apps Script 문법으로 작성해줘 (var 사용).
```

### 에러가 났을 때

**에러는 정상입니다!** 프로 개발자도 에러를 냅니다. 당황하지 마세요.

1. 에러 메시지 + 현재 코드를 **그대로 복사**
2. AI에게 보내면서: **"수정해줘. 뭐가 문제였는지 설명도 해줘"**
3. 로그(`Logger.log`)를 항상 남겨달라고 요청하면 문제 추적이 훨씬 쉬워집니다

### 결과가 예상과 다를 때

- 에러는 없지만 결과가 이상할 때
- **예상한 결과**와 **실제 결과**를 함께 AI에게 보내기
- "왜 다른지 확인하고 수정해줘"

### 개인정보 주의사항

- LLM에 로그를 보낼 때 실제 이름/이메일/전화번호는 **마스킹**
- Drive URL이나 폴더 ID는 **공유하지 마세요**
- 오늘 실습 데이터는 **모두 가짜**입니다

## Practice 1: 스프레드시트 정규화 (Beginner)

**예상 소요시간**: 약 10분

### 목표

50명 지원자 데이터를 정리하고 이메일을 매칭합니다.

### 입력 데이터

| 시트 | 컬럼 | 행 수 |
|------|------|-------|
| Sheet1 | 이름, 나이, 지원트랙, 지원동기, 연락처 | 50명 |
| Sheet2 | 이름, 이메일 | 42명 (8명 누락) |

### 문제점

- 이름에 불필요한 공백이 들어가 있음 (예: `"이  준"`)
- 연락처 형식이 제각각 (예: `010 1234 5678`, `010-1234-5678`, `+82-10-1234-5678`)
- Sheet2에 이메일이 없는 사람 8명 — 매칭 실패 처리 필요

### 프롬프트

아래 프롬프트를 ChatGPT 또는 Claude에 **그대로** 복사해서 입력하세요.

> **팁**: LLM에게 각 단계마다 로그(`Logger.log`)를 남겨달라고 요청하는 것을 권장합니다. 코드가 어떤 작업을 했는지 추적하기 쉬워집니다.

```
Google Apps Script로 스프레드시트 데이터를 정리하고 싶어.

현재 상황:
- 내 스프레드시트에 시트가 2개 있어
- Sheet1: 이름, 나이, 지원트랙, 지원동기, 연락처
- Sheet2: 이름, 이메일

문제점:
- 이름에 공백이 들어가 있는 경우가 있음 (예: "이  준")
- 연락처 형식이 제각각임 (010 1234 5678, 010-1234-5678, +82-10-1234-5678 등)
- Sheet2에 이메일이 있는데, 이름으로 매칭해서 Sheet1 데이터와 합치고 싶음

할 일:
1) 이름 공백 정리
2) 연락처를 010-1234-5678 형식으로 통일
3) Sheet2에서 이름으로 이메일 찾아서 매칭
4) 정리된 결과를 새 시트에 저장
5) 이메일 못 찾은 사람은 표시해줘
6) 몇 명 처리했는지 로그로 알려줘

Google Apps Script 문법으로 작성해줘 (var 사용).
```

### 실습 순서

1. 위 프롬프트를 복사해서 ChatGPT 또는 Claude에 붙여넣기
2. 받은 코드를 Apps Script 에디터에 붙여넣기
3. **▶ 실행** → 결과 시트 확인

### 기대 결과

- 새 시트 `"정규화_지원자_목록"` 생성
- 이름의 불필요한 공백 모두 정리
- 연락처 `010-xxxx-xxxx` 형식으로 통일
- Sheet2 이메일 매칭 완료
- 이메일 못 찾은 사람 `"매칭 실패"` 표시

```
│   └── ...
├── KDT/
├── Hackathon/
└── BootCamp/
```

각 문서 안에 지원자 이름, 트랙, 지원동기가 기록됩니다.

### 프롬프트

아래 프롬프트를 ChatGPT 또는 Claude에 **그대로** 복사해서 입력하세요.

```
Google Apps Script로 Drive에 폴더와 문서를 자동으로 만들고 싶어.

현재 상황:

4) 문서 안에 지원자 이름, 트랙, 지원동기를 적어줘
5) 같은 이름 파일이 이미 있으면 건너뛰고 로그만 남겨줘
6) 트랙별로 몇 개 만들었는지 요약해줘

Google Apps Script 문법으로 작성해줘 (var 사용).
```

### 실습 순서

1. 위 프롬프트를 복사해서 LLM에 붙여넣기
2. 받은 코드를 Apps Script 에디터에서 실행
3. **Google Drive**에서 `"트랙별_지원자"` 폴더가 생겼는지 직접 확인

## Practice 4: 72개 파일 자동 분류 (Capstone)

**예상 소요시간**: 약 15분
**선행 조건**: Practice 1~3 완료 후 사용 권장

### 목표

`GAS_Tutorial_Workspace` 폴더 안의 72개 파일을 태그 기반 규칙에 따라 자동으로 분류합니다.

### 태그 종류

| 분류 | 태그 |
|------|------|
| 프로젝트 | `[Hackathon]`, `[SeSAC]`, `[SSAFY]`, `[KDT]`, `[BootCamp]` |
| 외부 자료 | `[Ext]` |
| 연도 | `[2024]`, `[2025]` |
| 상태 | `[Draft]`, `[Final]`, `[Review]`, `[Archive]` |

### 분류 규칙

| 조건 | 이동 위치 |
|------|-----------|
| 프로젝트 태그 있음 | `Projects/해당프로젝트/` |
| `[Ext]` 태그 있음 | `External/` |
| 같은 이름 + 같은 파일 형식 중복 | `Duplicates/` |
| 태그 없음 / 이상한 파일 | `Unsorted/` |
| `"Copy of"`로 시작하는 파일 | `Unsorted/` |

### 프롬프트

아래 프롬프트를 ChatGPT 또는 Claude에 **그대로** 복사해서 입력하세요.

> **팁**: LLM에게 각 단계마다 로그(`Logger.log`)를 남겨달라고 요청하는 것을 권장합니다.

```
Google Apps Script로 엉망인 Drive 폴더를 자동 정리하고 싶어.

현재 상황:
- "GAS_Tutorial_Workspace" 폴더 안에 72개 파일이 여러 하위 폴더에 뒤섞여 있어
- 파일 이름에 대괄호 태그가 붙어 있어. 예: [SeSAC][2024][Final] 프로젝트_계획서

태그 종류:
- 프로젝트: [Hackathon], [SeSAC], [SSAFY], [KDT], [BootCamp]
- 외부 자료: [Ext]
- 연도: [2024], [2025]
- 상태: [Draft], [Final], [Review], [Archive]

할 일:
1) 프로젝트 태그가 있으면 → Projects/해당프로젝트/ 폴더로 이동
2) [Ext] 태그가 있으면 → External/ 폴더로 이동
3) 같은 이름+같은 타입 파일이 중복이면 → Duplicates/ 폴더로 이동
4) 태그가 없거나 이상한 파일 → Unsorted/ 폴더로 이동
5) "Copy of"로 시작하는 파일은 Unsorted로 보내줘
6) 다시 실행해도 이미 정리된 파일은 건너뛰게 해줘
7) 마지막에 폴더별로 몇 개 파일이 들어갔는지 요약해줘

Google Apps Script 문법으로 작성해줘 (var 사용).
```

### 기대 결과 구조

```
GAS_Tutorial_Workspace/
├── Projects/
│   ├── Hackathon/   (10개)
│   ├── SeSAC/       (8개)
│   ├── SSAFY/       (9개)
│   ├── KDT/         (8개)
│   └── BootCamp/    (6개)
├── External/        (8개)
├── Duplicates/      (6개)
└── Unsorted/        (17개)
```

**전체 합계: 72개**

코드를 실행한 뒤 결과가 예상과 다를 수 있습니다. 아래 케이스들을 점검해보세요.

## 에러 대응 템플릿

### 에러가 났을 때

아래 템플릿을 복사해서 에러 메시지와 코드를 채운 뒤 AI에게 보내세요.

```
방금 준 코드를 실행했는데 에러가 났어.

[에러 메시지]
여기에 실행 로그 에러를 그대로 붙여넣기

[현재 코드]
여기에 전체 코드를 붙여넣기

수정해줘. 뭐가 문제였는지 간단히 설명도 해줘.
```

### 결과가 예상과 다를 때

에러는 없지만 결과가 이상할 때 아래 템플릿을 사용하세요.

```
코드가 에러는 안 나는데 결과가 이상해.

[예상한 결과]
여기에 기대했던 결과 적기

[실제 결과]
여기에 실제 로그 붙여넣기

왜 다른지 확인하고 수정해줘.
```

## 마무리

### 오늘 배운 것

| 핵심 | 내용 |
|------|------|
| GAS란? | Google 자동화 도구 — 코드는 AI가 작성 |
| 프롬프트 공식 | 현재 상황 + 문제점 + 할 일을 구체적으로 |
| 에러 대응 | 에러 메시지를 AI에게 그대로 복사해서 전달 |
| 복잡한 문제 | 단계별로 쪼개서 해결 |

### GAS가 특히 유용한 경우

- 반복적이고 규칙이 있는 작업
- 여러 Google 서비스를 연결해야 하는 작업
- 많은 양의 데이터를 일괄 처리해야 하는 작업
- 정기적으로 자동 실행이 필요한 작업 (트리거 설정 가능)

### 다음 단계

GAS를 더 활용하고 싶다면 AI에게 이렇게 물어보세요:

- "Google Forms 응답이 오면 자동으로 이메일을 보내는 GAS 코드 만들어줘"
- "매일 오전 9시에 자동으로 실행되는 GAS 트리거 설정 방법 알려줘"
- "Google Sheets 데이터를 Gmail로 자동 발송하는 코드 만들어줘"

> **기억하세요**: 코드를 몰라도 됩니다. 무엇을 하고 싶은지 구체적으로 설명할 수 있으면, AI가 코드를 만들어줍니다.
