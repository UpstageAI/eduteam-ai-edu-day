Solution

도구: Google Apps Script로 엉망인 Drive 폴더를 자동 정리하고 싶어.

현재 상황:

  • "GAS_Tutorial_Workspace"에 72개 파일이 뒤섞여 있어
  • 파일 이름에 태그가 있어. 예: [SeSAC][2024][Final] 프로젝트_계획서
  • 태그: 프로젝트([Hackathon] 등), [Ext], [2024]/[2025], [Draft]/[Final] 등

할 일:

  1. 프로젝트 태그 → Projects/해당프로젝트/
  2. [Ext] 태그 → External/
  3. 같은 이름+타입 중복 → Duplicates/
  4. 태그 없는 파일 → Unsorted/
  5. "Copy of" → Unsorted로
  6. 이미 정리된 건 건너뛰기
  7. 폴더별 파일 수 요약

Google Apps Script 문법으로 작성해줘 (var 사용).
