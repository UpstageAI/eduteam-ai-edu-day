도구: Google Apps Script로 어떤 작업?

현재 상황:

  • 어떤 데이터를 사용?
  • 시트에 무슨 정보가 있는지?

할 일:

  1. 폴더 구조는?
  2. 문서에 뭘 넣을지?
  3. 중복 처리는?
  4. 결과 요약은?
Solution

도구: Google Apps Script로 Drive에 폴더와 문서를 자동으로 만들고 싶어.

현재 상황:

  • 아까 정리한 지원자 데이터가 스프레드시트에 있어
  • 시트에 이름, 지원트랙, 지원동기 등이 있어

할 일:

  1. Drive에 "트랙별_지원자" 폴더를 만들어줘
  2. 지원트랙별로 하위 폴더 만들기
  3. 지원자마다 "이름.docx" 문서 생성
  4. 문서 안에 이름, 트랙, 지원동기 기록
  5. 같은 이름 파일이 있으면 건너뛰기
  6. 트랙별 몇 개 만들었는지 요약

Google Apps Script 문법으로 작성해줘 (var 사용).
