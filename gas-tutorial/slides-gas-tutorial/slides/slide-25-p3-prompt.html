도구: Google Apps Script로 어떤 작업?

현재 상황:

  • 어떤 폴더? 어떤 상태?

할 일:

  1. 어디서 찾을지?
  2. 어떻게 보여줄지?
  3. 구분 방법은?
  4. 요약 정보는?
Solution

도구: Google Apps Script로 Drive 폴더 구조를 Sheet에 정리해서 보고 싶어.

현재 상황:

  • "GAS_Tutorial_Workspace" 폴더 안에 하위 폴더가 여러 개 있어
  • 어떤 구조인지 파악이 안 돼서 시트에 트리 형태로 보고 싶어

할 일:

  1. 폴더 이름으로 루트 폴더 찾기
  2. 하위 폴더/파일 전부 탐색 → 시트에 기록
  3. 들여쓰기로 깊이 표현
  4. 폴더/파일 구분
  5. 전체 경로 표시
  6. 폴더/파일 수 요약

Google Apps Script 문법으로 작성해줘 (var 사용).
