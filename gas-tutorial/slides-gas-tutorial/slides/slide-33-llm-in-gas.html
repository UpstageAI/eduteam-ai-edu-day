Bonus

GAS에서 LLM API를 쓰고 싶다면?

그냥 LLM에게 물어보세요!

"Google Apps Script에서 Solar API를 호출하고 싶어" 라고 말하면, GAS에 맞는 JavaScript 코드를 바로 알려줍니다.

GAS는 JavaScript 기반이기 때문에, LLM이 알아서 UrlFetchApp.fetch() 형태로 코드를 만들어줍니다.

더 정확한 코드를 원한다면?

공식 문서에서 API 사용법을 복사해서 LLM에게 함께 전달하세요. 문서 기반으로 더 정확한 코드를 생성합니다.

예: "이 API 문서를 참고해서 GAS 코드 만들어줘" + 공식 문서 코드 붙여넣기
